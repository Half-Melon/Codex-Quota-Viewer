## 交付物

- 一轮严格验收后的问题清单

## 约束

- 保持产品定位为**只读额度查看器**

- 菜单文本简洁、层级清晰、无明显冗余提示
- 源码中没有明显的切换器时代残留、死逻辑、误导命名或多余结构负担
- README 与当前真实产品行为一致
- `swift build` 通过
- `./scripts/build-app.sh` 通过
