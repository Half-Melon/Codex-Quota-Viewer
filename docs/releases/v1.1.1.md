# Codex Quota Viewer 1.1.1

## Highlights

- Adds the Windows tray version as an official release target.
- Ships Windows installer artifacts built from the Tauri tray app.
- Aligns Windows package metadata with the main `1.1.1` release version.

## Windows

The Windows version focuses on tray-first quota visibility and the core desktop actions already available in the repository:

- Show the current active Codex account quota from the system tray.
- Configure refresh behavior from a small settings window.
- Open the bundled Session Manager.
- Open the local Codex folder.
- Exit cleanly from the tray menu.
